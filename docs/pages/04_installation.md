`SRTsim` is implemented as an R package, which can be installed from GitHub by:

#### 1. Install `devtools` if necessary
```r
install.packages('devtools')
```

#### 2. Install `SRTsim`
```r
devtools::install_github('xzhoulab/SRTsim')
```
#### 3. Load package
```r
library(SRTsim)
```
